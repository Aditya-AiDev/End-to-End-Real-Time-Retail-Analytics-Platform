# CCTV Resource Folder

## Structure

```
CCTV/
├── footage/          ← Raw CCTV .mp4 clips (input to pipeline)
│   ├── CAM_1.mp4     → camera ID: cam1          (Entry camera)
│   ├── CAM_2.mp4     → camera ID: CAM2          (Main floor)
│   ├── CAM_3.mp4     → camera ID: PURPLLE_MUM_1076_CAM6  (Billing)
│   ├── CAM_4.mp4     → camera ID: CAM_FLOOR_02  (Floor 2)
│   └── CAM_5.mp4     → camera ID: CAM_FLOOR_03  (Floor 3)
│
├── dataset/          ← Reference data files
│   ├── POS_transactions.csv    ← Point-of-sale transaction records
│   ├── sample_events.jsonl     ← Sample event schema reference
│   └── Problem_Statement.pdf   ← Purplle Tech Challenge PS3
│
└── events_output/    ← Pipeline writes JSONL events here (auto-created)
    └── (generated by run.sh after processing footage)
```

## How to run the pipeline against these files

```bash
cd pipeline

# Windows (Git Bash / WSL):
./run.sh ST1076 "../CCTV/footage" http://localhost:8080

# The POS file is auto-discovered from the footage folder.
# Copy POS_transactions.csv into footage/ OR pass it separately:
copy CCTV\dataset\POS_transactions.csv CCTV\footage\
```
